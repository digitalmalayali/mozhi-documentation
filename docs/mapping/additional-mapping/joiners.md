# Joiners

Explicit joiners can be typed as follows:

|        | Mozhi  | Unicode Character(s)  | Malayalam Example | Mozhi Example |
| :----- | :----- | :-------------------- | :---------------- | :------------ |
| U+200D | ++     | ZERO WIDTH JOINER     | ന്‍                 | n`++          |
| U+200C | \|\|   | ZERO WIDTH NON JOINER | ൻ‌്ര                | n\|\|`ra      |
| U+200B | \|\|\| | ZERO WIDTH SPACE      | ൻ​െ               | n\|\|\|`e     |
